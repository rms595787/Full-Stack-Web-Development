# PostgreSQL – Sequences & Auto-Increment

## 🔢 Sequences
| Command | Explanation | Example |
|---------|-------------|---------|
| `CREATE SEQUENCE seq_name;` | Create a sequence | `CREATE SEQUENCE user_id_seq;` |
| `NEXTVAL('seq')` | Get next value | `SELECT NEXTVAL('user_id_seq');` |
| `CURRVAL('seq')` | Current value | `SELECT CURRVAL('user_id_seq');` |
| `SETVAL('seq', n);` | Reset/define current value | `SELECT SETVAL('user_id_seq', 100);` |
| `DROP SEQUENCE seq;` | Delete sequence | `DROP SEQUENCE user_id_seq;` |

---

## 🔄 Auto-Increment with SERIAL / IDENTITY
| Type | Explanation | Example |
|------|-------------|---------|
| `SERIAL` | Auto-increment integer (legacy style) | `id SERIAL PRIMARY KEY` |
| `BIGSERIAL` | Larger range (64-bit) | `id BIGSERIAL PRIMARY KEY` |
| `GENERATED ALWAYS AS IDENTITY` | Modern standard-compliant | `id INT GENERATED ALWAYS AS IDENTITY PRIMARY KEY` |
| `GENERATED BY DEFAULT AS IDENTITY` | Allows manual insert override | `id INT GENERATED BY DEFAULT AS IDENTITY` |

---

## 🛠 Example
```sql
-- Using SERIAL
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    name TEXT
);

-- Using IDENTITY
CREATE TABLE orders (
    id INT GENERATED ALWAYS AS IDENTITY PRIMARY KEY,
    total NUMERIC
);
