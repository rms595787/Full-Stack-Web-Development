# PostgreSQL – Connectivity (For Projects)

## 🌍 Connection String Formats
| Environment | Format | Example |
|-------------|--------|---------|
| **psql CLI** | `psql -h host -U user -d db` | `psql -h localhost -U dev -d company` |
| **URL (libpq style)** | `postgresql://user:pass@host:port/db` | `postgresql://dev:dev123@localhost:5432/company` |
| **Key-Value style** | `host=... user=... password=...` | `host=localhost user=dev password=dev123 dbname=company` |

---

## 🔌 Common Drivers
| Language | Library |
|----------|---------|
| Python | `psycopg2`, `asyncpg` |
| Node.js | `pg` |
| Java | JDBC (PostgreSQL driver) |
| Go | `pq`, `pgx` |
| PHP | PDO, pg_connect |

---

## 🛠 Example (Python + psycopg2)
```python
import psycopg2

conn = psycopg2.connect(
    dbname="company",
    user="dev",
    password="dev123",
    host="localhost",
    port=5432
)
cur = conn.cursor()
cur.execute("SELECT * FROM employees;")
print(cur.fetchall())
conn.close()
