# PostgreSQL – User & Role Management

## 🔑 Roles & Users
| Command | Explanation | Example |
|---------|-------------|---------|
| `CREATE ROLE role_name;` | Creates a new role (generic, not login) | `CREATE ROLE analyst;` |
| `CREATE USER user_name WITH PASSWORD 'pwd';` | Shortcut for role + login privilege | `CREATE USER dev WITH PASSWORD 'dev123';` |
| `ALTER ROLE role_name ...;` | Modify a role’s attributes | `ALTER ROLE dev WITH SUPERUSER;` |
| `DROP ROLE role_name;` | Deletes a role/user | `DROP ROLE dev;` |

---

## 🔐 Privileges & Grants
| Command | Explanation | Example |
|---------|-------------|---------|
| `GRANT privilege ON object TO role;` | Grants specific permission | `GRANT SELECT ON employees TO analyst;` |
| `REVOKE privilege ON object FROM role;` | Removes permission | `REVOKE INSERT ON employees FROM analyst;` |

---

## 👥 Role Membership
| Command | Explanation | Example |
|---------|-------------|---------|
| `GRANT role TO user;` | Assigns a role to another | `GRANT analyst TO dev;` |
| `SET ROLE role_name;` | Switch active role in session | `SET ROLE analyst;` |
