![List all databases](./_images/01.%20List%20all%20database.png)

2. Creating Database

   There are three ways through which you can create databases

   - command - `create database databaseName;`

   1. Run this command in terminal
   2. Run this command in pgAdmin4 > Query Tool
   3. Right click on database > create > database

3. Change a Database (when you want to work with `multiple databases` at the `same time` and `switch b/w them`)

   1. For Terminal

   - command - `\c <databaseName>`
   - example - `\c instaDB;`

   2. For pgadmin

   - click on database you want to work > re-open query tool

4. Deleting a Database

   - Command - `DROP DATABASE <dbname>;`

- when you try to delete a database you will error something like this -

  ERROR: database "test" is being accessed by other users

  DETAIL: There is 1 other session using the database.

- to handle this error either you have `force delete from pgadmin` or `disconnect database from all databases` then run same command again

## CRUD Operation

1. Create Table

   A table is a collection of related data held in a table format within a database.
   <img src="../03. Practical/_images/CRUD table.png" alt="Table" width="600">

   Syntax - create table TableName(column1 datatype(size), column2 datatype(size));

   Example - create table personDB (id INT, name VARCHAR(100), city VARCHAR(100));

2. Adding Data into Table

   Syntax:

   1. using column names and then asigning values

      - In this we could asign values to specific columns and if the order of columns is different than the table's structure there is no problem
      - But the values order assign to it should be according to the columns order mentioned
      - Syntax - `INSERT INTO personDB(id, name , city) VALUES ( 101, 'Raju', 'Delhi');`

   2. without using column names and just assigning values
      - In this order or values mapped to the table's structure, hence order it very important
      - Syntax - INSERT INTO personDB VALUES (102, 'Rahul');

3. Reading Data Table

   - Syntax

   1. SELECT \* FROM <table_name>;  
      Ex - select \* from peopleDB;
   2. SELECT <column_name> FROM <table_name>;  
      Ex - select id from peopleDB;

4. Modifying/Updating Data from a Table

   - Syntax  
     UPDATE <table_name>  
     SET colname = 'updated_Value';  
     WHERE colname=value;

5. Deleting Data from table
   - Syntax  
     Delete FROM <table_name>  
     WHERE colname=value;
