# Aggregate functions

An **aggregate function** in PostgreSQL performs a calculation on a set of values (multiple rows) and returns a single value as the result.

They are commonly used with the `GROUP BY` clause to group rows and summarize data.

`For more details checkout this file - ` [Aggregate Functions](../02.%20Theory/10.%20Aggregate%20Functions.md)

---

### Where do we need aggregate functions?

In situation like these:

- How to find total no. of employees?
- Employee with Max Salary
- Average salary of employees
