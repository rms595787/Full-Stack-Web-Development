# PostgreSQL psql Meta-Commands Cheat Sheet

This table lists the most commonly used **psql meta-commands (`\d`, `\dt`, etc.)** grouped by functionality.


## 1️⃣ Describe Database Objects
| Command | Usage / Description | Notes |
|---------|-------------------|-------|
| `\d` | List all tables, views, and sequences in the current database | Default list of accessible objects |
| `\d <table_name>` | Describe structure of a specific table, including columns and types | Shows column names, types, and modifiers |
| `\d+ <table_name>` | Describe table with additional info (size, description) | Useful for storage info |
| `\dS` | List all system tables | System catalog tables only |

## 2️⃣ List Specific Object Types
| Command | Usage / Description | Notes |
|---------|-------------------|-------|
| `\dt` | List all tables in current schema | Only tables, ignores views & sequences |
| `\dv` | List all views | Only views in current schema |
| `\ds` | List all sequences | Useful for auto-increment columns (`serial`) |
| `\di` | List all indexes | Indexes on tables, including primary keys |
| `\df` | List all functions | User-defined and built-in functions |
| `\dn` | List all schemas | Database namespaces |

## 3️⃣ Search / Filter Objects
| Command | Usage / Description | Notes |
|---------|-------------------|-------|
| `\d <pattern>` | Show objects matching pattern | Example: `\d emp*` shows tables starting with "emp" |
| `\dt <pattern>` | Show tables matching pattern | Example: `\dt public.*` lists all tables in public schema |
| `\dv <pattern>` | Show views matching pattern | Pattern matching similar to tables |

## 4️⃣ Connection & Info
| Command | Usage / Description | Notes |
|---------|-------------------|-------|
| `\conninfo` | Show current connection info | Displays host, port, database, and user |
| `\l` or `\list` | List all databases | Shows owner and encoding for each database |
| `\c <dbname>` | Connect to a database | Example: `\c mydb` switches database |

## 5️⃣ Help & Other Commands
| Command | Usage / Description | Notes |
|---------|-------------------|-------|
| `\?` | List all psql meta-commands | Interactive help |
| `\h <SQL_command>` | Show syntax of a SQL command | Example: `\h SELECT` |

---

### ✅ Notes:
1. Meta-commands **start with a backslash (`\`)** and are specific to `psql`.  
2. Use `\d` family for quick inspection of **tables, views, sequences, and indexes**.  
3. Filtering with patterns helps find objects without listing everything.  
4. Combine `\d+` for extra details like table size, indexes, and descriptions.
