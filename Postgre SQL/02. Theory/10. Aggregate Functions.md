# PostgreSQL Aggregate Functions

| S.No. | Function      | Usage / Meaning                                           | Example |
|-------|--------------|-----------------------------------------------------------|---------|
| 1.    | `COUNT()`    | Returns the number of rows (or non-NULL values).           | `SELECT COUNT(*) FROM employee;` |
| 2.    | `SUM()`      | Returns the total sum of a numeric column.                 | `SELECT SUM(salary) FROM employee;` |
| 3.    | `AVG()`      | Returns the average value of a numeric column.             | `SELECT AVG(salary) FROM employee;` |
| 4.    | `MIN()`      | Returns the smallest value in a column.                    | `SELECT MIN(salary) FROM employee;` |
| 5.    | `MAX()`      | Returns the largest value in a column.                     | `SELECT MAX(salary) FROM employee;` |
| 6.    | `STRING_AGG()` | Concatenates values into a single string (with delimiter). | `SELECT STRING_AGG(fname, ', ') FROM employee;` |
| 7.    | `ARRAY_AGG()` | Aggregates values into an array.                           | `SELECT ARRAY_AGG(fname) FROM employee;` |
| 8.    | `VARIANCE()` | Returns the variance of numeric values.                    | `SELECT VARIANCE(salary) FROM employee;` |
| 9.    | `STDDEV()`   | Returns the standard deviation of numeric values.          | `SELECT STDDEV(salary) FROM employee;` |
| 10.   | `BOOL_AND()` | Returns `TRUE` if all values are true.                     | `SELECT BOOL_AND(salary > 20000) FROM employee;` |
| 11.   | `BOOL_OR()`  | Returns `TRUE` if at least one value is true.              | `SELECT BOOL_OR(salary > 50000) FROM employee;` |
