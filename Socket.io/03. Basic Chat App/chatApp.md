# Creating Basic Chat App

1. Install package.json file
   - command `npm init -y`
2. Make changes in package.json file

```js
 "scripts": {
    "start": "node index.js",
    "dev": "nodemon index.js"
  },
```

3. Install dependencies nodemon, express, socket.io
   - command - `npm i express nodemon socket.io`
4. Run the server
   - command - `npm run dev`
