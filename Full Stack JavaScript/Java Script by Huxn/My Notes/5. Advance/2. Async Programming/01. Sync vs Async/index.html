Learnings Covered

Synchronous JavaScript
- Line by Line execution of code, here code is blocked(meaning untill first code is 
completed next could not be executed)

Asynchronous JavaScript
- Here code is not blocked, if any code taking more time then we move on to next 
operation and perform it first and then performs the previous operation
- we do such kind of things with the help of techniques like callbacks, promises, 
and async/await, which allow developers to handle asynchronous operations more 
efficiently
