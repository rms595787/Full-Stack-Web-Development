Learnings Covered

Async Function
Async is a special function that is designed to operate asynchronously, meaning that 
it can perform tasks in the background while other code continues to execute. Async 
functions are marked with the async keyword.

Async
Async is a keyword use to turn function declaration to async function.


Await
The await operator is used to wait for a promise. It can only be used inside an 
async function within regular JavaScript code.
