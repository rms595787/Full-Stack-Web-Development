# OOPS

> Object-Oriented Programming(OOP) is a programming **_language model/style_** organized around objects rather then "actions" and data rather then logic.

# JS

> JavaScript is a versatile and dynamic programming language that supports multiple paradigms, including procedural, functional, and object-oriented programming. While JavaScript **_is not a purely object-oriented language_** like java or C++, it provides the tools and features to implement OOP conecepts effectively.
