            1.      reduce() method
            
            1.1.    The "reduce" method executes a reduce function for array element.
            1.2.    The "reduce" method returns a single value: the function's accumulated result.
            1.3.    The "reduce" method does not execute the function for empty array elements.
            1.4.    The "reduce" method does not change the original array.
