Full Stack Java Script

Learnings Covered

            1.  Using Map() constructor to create map data structure
            2.  Using map.set() -> To add keys and values in  map
            3.  Using map.keys() -> To check the keys
            4.  Using map.values() -> To check the values
            5.  Using map.size -> To check the size of the map
            6.  Using map.delete() -> To delete the key and value pair
            7.  Using for(let [keys,values] of map) ->To iterate the keys and values from the map data structure
            8.  Using map.get() -> To get the keys and values from the map
            9.  Using map.has() -> To check has the keys and values from the map
            10. Using map.clear() -> To delete all keys and values from the map and left the map empty
