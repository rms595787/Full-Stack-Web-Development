## 🧠 What I Learned

Range inputs look simple but can be **customized heavily using browser-specific selectors**.

This project explores:

```id="a19p9e"
::-webkit-slider-runnable-track
::-webkit-slider-thumb
```

These selectors allow developers to **style the slider track and thumb separately**, giving complete control over the UI.

Understanding these selectors is important when designing:

- volume controls
- brightness sliders
- progress controls
- custom UI components

It also introduced me to the concept of **WebKit engine styling**, which powers browsers like Chrome and Safari.
