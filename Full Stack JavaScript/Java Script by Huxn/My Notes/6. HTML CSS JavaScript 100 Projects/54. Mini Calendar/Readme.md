## 🧠 What I Learned

In this project I explored how JavaScript can format dates in different ways depending on the user's region.

Using functions like:

```id="g76y9g"
toLocaleDateString()
toLocaleTimeString()
```

developers can display dates automatically formatted for different countries.

This is extremely important for applications that serve **international audiences**, ensuring users see dates in familiar formats.
