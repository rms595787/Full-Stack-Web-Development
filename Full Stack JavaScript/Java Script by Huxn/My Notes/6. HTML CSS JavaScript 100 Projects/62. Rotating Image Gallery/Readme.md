## 🧠 What I Learned

This project helped me explore **advanced 3D transformations in CSS** by combining **perspective with transform properties**.

The key idea is that when perspective is applied to a parent element, child elements can move and rotate in **three-dimensional space** using the **X, Y, and Z axes**.

Some important transform functions used in 3D effects include:

```css
transform: rotateX(45deg);
transform: rotateY(30deg);
transform: translateZ(100px);
```

### Understanding the Axes

- **rotateX()** → tilts the element forward or backward
- **rotateY()** → rotates the element left or right
- **rotateZ()** → rotates the element like a wheel

Another important property learned in this project is:

```css
transform-style: preserve-3d;
```

Normally, child elements inside a transformed element are flattened into the same plane.
Using `preserve-3d` allows child elements to **maintain their own 3D positioning**, enabling more complex structures like **3D cubes, cards, or layered animations**.

When these properties are combined with perspective, we can build impressive visual effects such as:

- 3D card flips
- rotating cubes
- layered animations
- interactive UI components

This project showed how **a few CSS properties can transform flat layouts into immersive 3D experiences**, making interfaces feel more dynamic and modern.
