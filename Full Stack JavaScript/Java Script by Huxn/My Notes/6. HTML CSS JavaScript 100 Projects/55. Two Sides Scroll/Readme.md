## 🧠 What I Learned

This project explores **HTML entities**, which represent special characters using numeric codes.

Example:

```id="5wtx0k"
&#8595;
```

This renders a **down arrow symbol**.

I also explored **URL parameters**, which allow applications to pass information inside links.

Example:

```id="gty3ch"
?direction=up
?direction=down
```

Using `URLSearchParams`, developers can read these values and modify application behavior accordingly.

I also experimented with `clientHeight`, which returns the visible height of an element and helps when building **sliders or scroll-based components**.
