## 🧠 What I Learned

APIs allow applications to **fetch dynamic content from external sources**.

In this project I used the image API:

```id="q3s0u8"
https://picsum.photos/1600/900/?random
```

Each request generates a completely new image.

By combining this with carousel logic, the project creates a **dynamic slideshow that always displays fresh images**.

This approach is useful for building:

- image galleries
- hero sliders
- portfolio showcases

without storing large image collections locally.
