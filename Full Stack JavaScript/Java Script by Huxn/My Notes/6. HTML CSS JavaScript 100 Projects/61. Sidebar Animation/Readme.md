## 🧠 What I Learned

In this project I explored **CSS perspective**, which allows developers to create **3D visual effects on web elements**.

Normally, elements on a webpage exist in a **2D space** using only the **X and Y axes**. By introducing perspective, the browser simulates a **third dimension (Z-axis)**, making elements appear closer or farther from the viewer.

Example syntax:

```css
container {
  perspective: 1000px;
}
```

The perspective value controls how strong the 3D effect appears:

- **Smaller values** → stronger depth effect
- **Larger values** → flatter appearance

Perspective works together with the `transform` property. For example:

```css
.card {
  transform: rotateY(45deg);
}
```

When a parent container has perspective applied, the element appears to **rotate in actual 3D space instead of flat rotation**.

This concept is widely used in modern UI designs such as:

- 3D card flips
- product showcases
- interactive landing page animations
- immersive UI effects

One of the most exciting things about working with perspective is that **small creative ideas can produce visually impressive results with very little code**.
