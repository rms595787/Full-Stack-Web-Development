## 🧠 What I Learned

Positioning plays a major role in building modern UI experiences.

This project explores the power of:

```id="wy1l2b"
position: fixed
```

Unlike normal elements, fixed elements stay **anchored to the viewport even when the page scrolls**.

This technique is widely used for:

- sticky navigation bars
- floating action buttons
- live chat widgets
- notification banners

I also experimented with **scroll events** using JavaScript:

```id="sre35p"
window.addEventListener("scroll")
```

By comparing values like `scrollY`, `offsetHeight`, and page height, developers can build effects such as:

- scroll progress bars
- dynamic headers
- reveal animations
- infinite scrolling

This project shows how combining **CSS positioning with JavaScript scroll events** can create powerful interactive interfaces.
