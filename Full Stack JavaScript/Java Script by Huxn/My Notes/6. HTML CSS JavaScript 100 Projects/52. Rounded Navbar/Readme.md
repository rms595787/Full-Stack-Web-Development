## 🧠 What I Learned

This project explores **CDN libraries and advanced CSS selectors**.

Using **cdnjs**, I integrated **Font Awesome icons** without downloading any files locally.

CDNs allow developers to quickly use popular libraries by simply linking them in HTML.

I also explored multiple ways of selecting child elements in CSS such as:

```id="c3q2mw"
.parent > div
.parent div
li:nth-child(2)
div:first-child
div:last-child
```

Understanding these selectors allows developers to **target elements precisely without adding unnecessary classes**.
