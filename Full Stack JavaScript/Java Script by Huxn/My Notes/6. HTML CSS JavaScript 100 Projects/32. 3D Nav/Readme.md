## 🧠 What I Learned

- The **CSS perspective** property and how it creates 3D depth.
- How developers use it in:
  - rotating cards
  - 3D navigation
  - product showcases
  - VR-like UIs

- Learned child selectors in CSS like:
  - `nth-child()`
  - `first-child`
  - `last-child`
  - `nth-of-type()`
  - And how they help create complex animations easily.
