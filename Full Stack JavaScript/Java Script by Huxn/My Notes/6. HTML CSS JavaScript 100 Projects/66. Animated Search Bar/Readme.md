## 🧠 What I Learned

This project focuses on how **small CSS properties and external libraries can create polished user interfaces**.

One of the main things explored in this project is **Font Awesome**, a popular icon library used in web development.

Instead of using images for icons, Font Awesome provides **scalable vector icons** that can be easily inserted into HTML.

Example usage:

```html
<i class="fa-solid fa-heart"></i>
```

To use Font Awesome, developers often include it through **CDNJS**, which allows loading libraries directly from a remote server without downloading files locally.

Example CDN link:

```html
https://cdnjs.cloudflare.com
```

Using a CDN has several advantages:

* Faster loading due to caching
* No need to store large files locally
* Easy integration into projects

Another key concept explored in this project is **CSS transitions**.

Transitions allow properties to change **smoothly over time instead of instantly**.

Example:

```css
transition: all 0.3s ease;
```

Common properties that work well with transitions include:

* opacity
* transform
* color
* background-color
* visibility

For example, combining **visibility** with **opacity** can create smooth show/hide animations:

```css
opacity: 0;
visibility: hidden;
```

Then when activated:

```css
opacity: 1;
visibility: visible;
```

Even though these are very simple CSS properties, combining them creatively can produce **beautiful animations and interactive UI elements**.

This project reinforced an important lesson: sometimes **the most impressive interfaces are built from the simplest properties used creatively**.
