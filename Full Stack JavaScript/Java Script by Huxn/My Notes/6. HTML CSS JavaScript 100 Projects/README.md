# 🚀 100 JavaScript Projects Challenge

> A collection of **100 frontend projects built using HTML, CSS, and JavaScript** to strengthen my understanding of **UI design, DOM manipulation, animations, API integration, and responsive web development**.
>
> Each project focuses on learning a **specific concept or feature used in real-world web applications**.

---

## 🧠 What This Repository Covers

Throughout these projects I practiced:

* DOM Manipulation
* CSS Animations & Transitions
* Responsive Web Design
* Flexbox & Grid Layout
* Event Handling
* Form Validation
* JavaScript Logic Building
* API Integration
* UI Components
* Interactive Web Applications
