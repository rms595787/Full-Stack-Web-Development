## 🧠 What I Learned

Mouse tracking opens the door to highly interactive UI designs.

Using the `mousemove` event, developers can track cursor position in real time.

Properties like:

- `e.clientX`
- `e.clientY`

provide exact cursor coordinates within the viewport.

By adding slight delays or transitions, these coordinates can be used to create effects such as:

- cursor trails
- parallax animations
- interactive hover elements

Even small mouse interactions can dramatically improve the **feel of an interface**.
