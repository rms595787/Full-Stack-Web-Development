## 🧠 What I Learned

In this project I learned how **JavaScript can fully control and manipulate the DOM**, allowing us to create, update, and remove elements dynamically without writing additional HTML.

One of the main concepts was **creating elements directly with JavaScript**.

Example:

```javascript
document.createElement("li");
```

This allows us to generate new HTML elements dynamically, such as new todo items.

After creating elements, they can be added to the page using:

```javascript
parent.appendChild(child);
```

This attaches the newly created element to an existing container in the DOM.

Similarly, elements can be removed when needed using:

```javascript
parent.removeChild(child);
```

This is useful for features like **deleting completed tasks**.

I also worked with other useful DOM properties like:

```javascript
element.textContent;
element.innerHTML;
```

to update the content of elements.

Another important concept was **class manipulation**, which helps control styles and behavior.

Examples include:

```javascript
element.classList.add("active");
element.classList.remove("active");
element.classList.toggle("completed");
```

Using these methods, JavaScript can dynamically update the UI.

### Key Learning

This project showed how **JavaScript alone can manage the entire structure of a webpage**, which is the fundamental idea behind modern frameworks like **React, Vue, and Angular** where the UI is generated dynamically.
