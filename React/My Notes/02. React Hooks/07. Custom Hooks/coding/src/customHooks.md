# Custom Hooks

Custom Hooks are JavaScript functions that start with the prefix use ( e.g., useFetch, useForm ) and can call other hooks within them. They allow you to `extract and reuse` `logic` that involes state or side effects, making your components more readable and maintainable.

# useName

To create a new custom hook you have to follow this convention

- you have to use `use` before naming the convention
- Ex - useName, useFetch
