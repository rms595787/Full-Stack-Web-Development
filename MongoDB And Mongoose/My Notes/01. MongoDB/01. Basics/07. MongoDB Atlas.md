# MongoDB Atlas

**MongoDB Atlas** is a **cloud-based database service** provided by MongoDB.

- It lets you run **MongoDB clusters** on cloud providers like **AWS, Azure, and GCP**.
- You don’t need to install or manage servers yourself → Atlas handles **scaling, backups, monitoring, and security automatically**.
- It offers a **free tier** (great for learning and small apps).

👉 In short: **MongoDB Atlas = MongoDB on the cloud, fully managed**.
