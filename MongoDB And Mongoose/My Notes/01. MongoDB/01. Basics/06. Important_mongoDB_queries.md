# 📌 Important MongoDB Queries (SQL vs MongoDB Cheatsheet)

## 🗄️ Database Commands

| **SQL** | **MongoDB** |
|---------|-------------|
| `CREATE DATABASE mydb;` | `use mydb` |
| `DROP DATABASE mydb;` | `db.dropDatabase()` |
| `SHOW DATABASES;` | `show dbs` |

---

## 📂 Table vs Collection

| **SQL** | **MongoDB** |
|---------|-------------|
| `CREATE TABLE users (...);` | `db.createCollection("users")` |
| `DROP TABLE users;` | `db.users.drop()` |
| `SHOW TABLES;` | `show collections` |

---

## ✍️ Insert

| **SQL** | **MongoDB** |
|---------|-------------|
| `INSERT INTO users (name, age) VALUES ('Rahul', 23);` | `db.users.insertOne({name: "Rahul", age: 23})` |
| Insert multiple rows | `db.users.insertMany([{name: "A"}, {name: "B"}])` |

---

## 🔍 Select (Find)

| **SQL** | **MongoDB** |
|---------|-------------|
| `SELECT * FROM users;` | `db.users.find()` |
| `SELECT name, age FROM users;` | `db.users.find({}, {name:1, age:1})` |
| `SELECT * FROM users WHERE age = 23;` | `db.users.find({age: 23})` |
| `SELECT * FROM users WHERE age > 20;` | `db.users.find({age: {$gt: 20}})` |
| `SELECT * FROM users WHERE age BETWEEN 20 AND 30;` | `db.users.find({age: {$gte: 20, $lte: 30}})` |
| `SELECT * FROM users WHERE name LIKE 'Ra%';` | `db.users.find({name: /^Ra/})` |
| `SELECT * FROM users ORDER BY age DESC;` | `db.users.find().sort({age: -1})` |
| `SELECT * FROM users LIMIT 5;` | `db.users.find().limit(5)` |
| `SELECT * FROM users OFFSET 5 LIMIT 5;` | `db.users.find().skip(5).limit(5)` |
| `SELECT COUNT(*) FROM users;` | `db.users.countDocuments()` |
| `SELECT DISTINCT(city) FROM users;` | `db.users.distinct("city")` |

---

## ✏️ Update

| **SQL** | **MongoDB** |
|---------|-------------|
| `UPDATE users SET age=24 WHERE name='Rahul';` | `db.users.updateOne({name: "Rahul"}, {$set: {age: 24}})` |
| Update multiple | `db.users.updateMany({city: "Delhi"}, {$set: {verified: true}})` |

---

## 🗑️ Delete

| **SQL** | **MongoDB** |
|---------|-------------|
| `DELETE FROM users WHERE name='Rahul';` | `db.users.deleteOne({name: "Rahul"})` |
| Delete multiple | `db.users.deleteMany({city: "Delhi"})` |
| `TRUNCATE TABLE users;` | `db.users.deleteMany({})` |

---

## 🔗 Joins

| **SQL** | **MongoDB ($lookup)** |
|---------|-----------------------|
| `SELECT users.name, orders.amount FROM users JOIN orders ON users.id = orders.user_id;` | ```js db.users.aggregate([ { $lookup: { from: "orders", localField: "id", foreignField: "user_id", as: "orders" } } ]) ``` |

---

## 📊 Group By / Aggregation

| **SQL** | **MongoDB** |
|---------|-------------|
| `SELECT city, COUNT(*) FROM users GROUP BY city;` | ```js db.users.aggregate([ { $group: { _id: "$city", count: { $sum: 1 } } } ]) ``` |
| `SELECT city, AVG(age) FROM users GROUP BY city;` | ```js db.users.aggregate([ { $group: { _id: "$city", avgAge: { $avg: "$age" } } } ]) ``` |

---

⚡ **Key Differences**:
- SQL → Relational (tables, rows, columns)  
- MongoDB → Document-oriented (collections, JSON-like documents)  
- SQL Joins → MongoDB `$lookup` in Aggregation  
- SQL Schema → MongoDB is schema-less (flexible)
