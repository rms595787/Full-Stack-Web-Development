# Setup in mac

1. Download MongoDB Compass (GUI) - [LINK](https://www.mongodb.com/try/download/compass)

2. Go to MongoDB Installation Documentation - [LINK](https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/)

   - Run these bash commands

   ```bash
   <!-- 1st command to run -->
   xcode-select --install

   <!-- 2nd command to run -->
   brew tap mongodb/brew

   <!-- 3rd command to run -->
   brew update

   <!-- 4th command to run -->
   brew install mongodb-community@8.0
   ```

3. Commands to run and stop mongodb

```bash
# To run MongoDB (i.e. the mongod process) as a macOS service, run:

brew services start mongodb-community@8.0

# To stop a mongod running as a macOS service, use the following command as needed:

brew services stop mongodb-community@8.0
```

4. To verify that MongoDB is running, perform one of the following:

- If you started MongoDB as a macOS service:

```bash
brew services list
```

5. Connect and Use MongoDB

- To begin using MongoDB, connect mongosh to the running instance. From a new terminal, issue the following:

```bash
mongosh
```
