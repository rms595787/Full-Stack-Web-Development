## 3. UPDATE

Available Functions

- updateOne()

  1.  To `Adds a new field or updates the value of an existing field`

      `$set` is a pre-defined operator - Adds a new field or updates the value of an existing field.

      Syntax

      Example 1

      ```js
      db.cars.updateOne({ model: "Nexon" }, { $set: { color: "Red" } });
      ```

        <img src="./_images/updateOne_set.png" width="700"/>

  2.  To `Removes a field from a document`

      `$unset` is a pre-defined operator - Removes a field from a document.

      Example 2

      Syntax

      ```js
      db.cars.updateOne({ model: "Nexon" }, { $unset: { color: "" } });
      ```

        <img src="./_images/updateOne_unset.png" width="700"/>

  3.  To `Add element to an array`

      `$push`

      - Appends a value to an array fiel.
      - If the field doesn’t exist, MongoDB creates it as an array.

      Example 3

      Syntax

      ```js
      db.cars.updateOne(
        { model: "Nexon" }, // Filter to find the specific document
        { $push: { features: "Heated Seats" } }
      );
      ```

        <img src="./_images/updateOne_push.png" width="700"/>

  4.  To `Remove element from an array`

      `$pull` - Removes all matching values from an array.

      Example 4

      Syntax

      ```js
      db.cars.updateOne(
        { model: "Nexon" },
        { $pull: { features: "Heated Seats" } }
      );
      ```

        <img src="./_images/updateOne_pull.png" width="700"/>

  5.  To `push multiple values at once` we need to use `$each`
      Note - `$each` only works with `$push` or `$addToSet`  
      `$push` with `$each` is a pre-defined operator - push multiple values

      Syntax

      ```js
      db.cars.updateOne(
        { model: "Nexon" }, // Filter to find the specific document
        {
          $push: {
            features: { $each: ["Wireless Charging", "Voice Control"] },
          },
        }
      );
      ```

        <img src="./_images/updateOne_pull.png" width="700"/>

  6.  To `pull multiple values at once` we need to use `$in`

      `$pull` with `$in` is a pre-defined operator - pull multiple values

      Syntax

      ```js
      db.cars.updateOne(
        { model: "Nexon" },
        { $pull: { features: { $in: ["Wireless Chargin", "Voice Control"] } } }
      );
      ```

        <img src="./_images/updateOne_pull_in.png" width="700"/>

  7.  To `update - if present and insert - if not present`

      `$upsert` is a combination of the operations `"update"` and `"insert"`

      Syntax

      ```js
      db.students.updateMany(
        { name: "Jane Doe" },
        { $set: { age: 22, course: "Mathematics" } },
        { upsert: true }
      );

      db.cars.updateMany(
        { model: "Venue" },
        { $set: { maker: "Hyundai" } },
        { upsert: true }
      );
      ```

        <img src="./_images/updateMany_upsert.png" width="700"/>

- updateMany()

  Syntax

  ```js
  db.cars.updateMany(
    { model: "Nexon" }, // Filter to find all documents where model is Nexon
    { $set: { fuel_type: "Electric" } } // update the fuel_type to Electric
  );

  db.cars.updateMany({}, { $set: { color: "Blue" } });
  ```

    <img src="./_images/updateOne_pull_in.png" width="700"/>
