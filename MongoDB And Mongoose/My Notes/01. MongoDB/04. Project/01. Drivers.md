## Todo List python project with mongodb

Steps to setup python with mongodb

1. Click on the link or go to mongodb drivers
2. Choose python as your language - [Link](https://www.mongodb.com/docs/languages/python/) You will be able to see a driver named `PyMongo`
3. Go to the documentation of PyMongo Driver, click on Getting started with PyMongo [Link](https://www.mongodb.com/docs/languages/python/pymongo-driver/current/get-started/#std-label-pymongo-get-started)
4. Basic code will be available there follow that

   ```py
   from pymongo import MongoClient

   uri = "<connection string URI>"
   client = MongoClient(uri)

   try:
       database = client.get_database("sample_mflix")
       movies = database.get_collection("movies")

       # Query for a movie that has the title 'Back to the Future'
       query = { "title": "Back to the Future" }
       movie = movies.find_one(query)

       print(movie)

       client.close()

   except Exception as e:
       raise Exception("Unable to find the document due to the following error: ", e)

   ```

5. For connecting with mongoDB you have to provide `uri` either from localhost or from mongodb atlas
6. For local connection use this command
   ```py
   uri = "mongodb://localhost:27017/"
   ```
