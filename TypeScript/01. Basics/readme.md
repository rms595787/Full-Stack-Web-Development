# TypeScript

"JavaScript with Superpowers": TypeScript is like a more powerful version of JavaScript. It lets you do everything that JavaScript does, but it adds some extra features to make your life easier when writing larger and more complex programs.

[TypeScript Original Website](https://www.typescriptlang.org)

# To work with TypeScript you must have these things in your machine

- IDE - vs code or any other ide
- Node.js setup
- TypeScript - install using command( npm install -g typescript )
- run this command for easy execution of typescript ( npm i -g ts-node )

# Commant To run TypeScript file

- file name -> index.ts
- first command
- - tsc index.ts
- This will convert you typescript file to JavaScript file

- second command
- - node index.js
- now our file will be executed
