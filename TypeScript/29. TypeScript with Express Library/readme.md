## Working with express library in typescript

- run these commands in your terminal

```js
// syntax
npm init -y // this will generate a package.json file
npm i axios // this will install axios library
npm i express // this will install express library
```

- go to package.json edit "scripts"
- remove default script and paste this line in your script
- "scripts":{
  "start":"ts-node /Users/rahulsh9956/Documents/Coding/Full-Stack-Web-Development/TypeScript/29. TypeScript with Express Library/index.ts"
  }

- to run the index.ts
- use command "npm start" in your terminal
- make sure to be in the same folder where all node_module is installed before starting the npm
- go to [localhost:3000/](http://localhost:3000)
